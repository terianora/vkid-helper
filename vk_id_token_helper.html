VK ID Token Helper

Эта страница нужна только для первого логина. Она запускает VK ID flow, получает code и device_id, обменивает их на токены и собирает готовый блок vk_auth для вашего автопостера.

Важно: в настройках приложения VK ID должен быть добавлен redirect URL, который вы укажете ниже. Для локального запуска удобно использовать http://127.0.0.1:8765/vk_id_token_helper.html.

1. Настройки

Заполните APP_ID и redirect URL, затем нажмите «Войти через VK ID».

2. Результат

После успешной авторизации сюда появится готовый JSON-блок для вставки в config.json.

3. Что дальше

1. Скопируйте блок vk_auth в ваш config.json.

2. Уберите старые значения из access_token, photo_access_token и wall_access_token, либо оставьте их пустыми.

3. После первого успешного запуска скрипт сам будет обновлять access_token по refresh_token.
